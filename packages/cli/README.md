# QualWeb CLI

## How to install

```shell
  $ npm i -g @qualweb/cli
```

## How to run

### Normal evaluation

```shell
  $ qw -u https://act-rules.github.io/pages/about/
```

### Evaluation with EARL report

```shell
  $ qw -u https://act-rules.github.io/pages/about/ -r earl
```
